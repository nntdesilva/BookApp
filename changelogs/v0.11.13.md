# Version 0.11.13 - 2026-03-19

## Bug Fix: App Runner VPC Connector for Analysis and Chat

CD workflow (`.github/workflows/cd.yml`) now attaches a VPC connector for egress on the **analysis-service** and **chat-service** App Runner deployments. `aws apprunner update-service` network configuration includes `EgressConfiguration` with `EgressType: VPC` and `VpcConnectorArn` from the new `VPC_CONNECTOR_ARN` GitHub secret. Both deploy jobs validate that the secret is present before deploy. Fixes outbound connectivity (e.g. Redis, inter-service URLs) for services that run with private ingress.
