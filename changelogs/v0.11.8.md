# Version 0.11.8 - 2026-03-17

## Bug Fix: Word Search Tools Only for Explicit Word Count Requests

Tightened the AI prompt and tool descriptions in `chat-service/services/aiService.js` so that `resolve_book_for_search` and `count_word_in_book` are invoked only when the user explicitly asks to count words or phrases in a book's text. The AI will no longer call these tools for general informational queries (plot, author, themes, background, history, or similar). This reduces unnecessary Gutenberg API calls and improves response quality for non–word-count questions.
