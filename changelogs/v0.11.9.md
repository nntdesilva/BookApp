# Version 0.11.9 - 2026-03-17

## Refactor: Remove resolve_book_for_search Tool

Removed the separate `resolve_book_for_search` tool from the chat AI. Book resolution is now handled internally within `count_word_in_book` and `count_related_words_in_book`, so the AI calls these tools directly with the book title and search term. This simplifies the tool flow (one call instead of two) and reduces Gutenberg API round-trips.
