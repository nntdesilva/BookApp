# Version 0.4.0 - 2026-01-28

## Major Feature: Intelligent Colored Badge System

This release introduces an intelligent badge system that automatically highlights book titles in search results with colored badges, distinguishing between series queries and standalone books.

## What's New

### Colored Badge System

- **Series Query Detection**: AI automatically detects when users search for book series
- **Automatic Badge Assignment**: Books receive colored badges (red, blue, green, yellow) based on search context
- **Visual Distinction**: Users can easily identify which books belong to a series vs. standalone books
- **Intelligent Matching**: Badge system preserves search intent through title correction logic

### Technical Implementation

**New Features:**

- Series query detection algorithm
- Badge color assignment logic
- Title correction with search intent preservation
- Visual badge rendering in search results

### User Experience

- **Clear Visual Feedback**: Color-coded badges make it easy to scan search results
- **Series Recognition**: Quickly identify books that are part of a series
- **Consistent Badging**: Reliable badge assignment based on query context
