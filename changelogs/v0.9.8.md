# Version 0.9.8 - 2026-02-24

## UI Redesign: Black & White with Refined Book Tag Colours

A full visual redesign of `public/css/style.css`. The aesthetic shifts to a clean, minimalist palette — white background, near-black text, hairline grey borders — evoking a timeless, editorial sensibility. All typography (Playfair Display + Cormorant Garamond), layout measurements, and animations are unchanged.

## What's New

### Global Palette

| Element | Before | After |
|---|---|---|
| Page background | `#f8f6f1` (warm cream) | `#ffffff` (pure white) |
| Primary text | `#1a1a1a` | `#0a0a0a` (near black) |
| Secondary text | `#6b6b6b` | `#6b6b6b` (unchanged) |
| Borders (default) | `#d4c5a9` (tan) | `#e0e0e0` (light grey) |
| Borders (focused) | `#1a1a1a` | `#0a0a0a` |
| Card/surface backgrounds | `#ffffff` | `#ffffff` (unchanged) |

### Buttons

All filled buttons (search, send, auth, home) are jet black (`#0a0a0a`) with white text. Hover state darkens slightly to `#3a3a3a`. Ghost buttons (logout, search-again) use a light grey border at rest and invert to black fill on hover.

### Decorative Elements

- Quote mark on book cards: `#d4c5a9` (tan) → `#d4d4d4` (soft grey)
- Welcome message icon: `#d4c5a9` → `#d4d4d4`
- Scrollbar thumb: tan tint → `rgba(0, 0, 0, 0.15)`, deepens to `rgba(0, 0, 0, 0.35)` on hover

### Book Tag Colour System

Tags are now visually distinct with muted, low-opacity colour tints that remain legible without competing with the monochrome layout.

| Tag type | Background | Text colour | Underline |
|---|---|---|---|
| Original (searched) book | `rgba(123, 179, 46, 0.1)` — soft green | `#4e7a10` | `rgba(123, 179, 46, 0.4)` |
| Book in series | `rgba(224, 192, 40, 0.12)` — warm yellow | `#7a6008` | `rgba(224, 192, 40, 0.45)` |
| Unrelated book | `rgba(242, 104, 34, 0.07)` — light orange | `#c8541a` | `rgba(242, 104, 34, 0.25)` |

The same colour logic applies to both `.book-tag` and `.book-badge` variants.

### Chat Interface

- User message bubble: black background, white text (unchanged feel, now consistent with new button style)
- Assistant message bubble: white background, `#1a1a1a` text, `#e0e0e0` border
- Error message: `#f8f8f8` background, `#3a3a3a` text, `#e0e0e0` border
- Input focus border: `#0a0a0a`

### Auth Pages

- Form inputs: `#e0e0e0` border at rest, `#0a0a0a` on focus
- Auth button: black fill, white text; hover `#3a3a3a`
- Footer links: black text, light grey underline; hover underline turns black

## Files Changed

- **`public/css/style.css`** — full palette replacement (~50 colour value updates); no structural or typographic changes

---

All existing features (book search, chat, favourites, word count, semantic search, statistics, visualizations) are fully unchanged.
