# Version 0.10.0 - 2026-02-26

## New Feature: MCP Server for Cursor

All book-app tools are now exposed as a standalone [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) server, making them directly available inside Cursor as native AI tools.

## What's New

### MCP Server (`mcp-server.js`)

A new entry-point `mcp-server.js` runs the book-app tool suite over stdio transport using `@modelcontextprotocol/sdk`. Cursor (or any MCP-compatible client) can invoke all nine tools without going through the web application.

The server connects to MongoDB on startup, pre-resolves the configured user, and redirects `console.log` to stderr so the stdio protocol stream remains clean.

### Tools Exposed

#### Favourites (4 tools)

| Tool | Description |
|---|---|
| `add_to_favorites` | Add a book by ISBN-13 and title |
| `remove_from_favorites` | Remove a book by ISBN-13 |
| `list_favorites` | List all books in the favourites list |
| `remove_all_favorites` | Clear the entire favourites list at once |

#### Gutenberg Word Search (2 tools)

| Tool | Description |
|---|---|
| `resolve_book_for_search` | Check if a public-domain book is available in Project Gutenberg |
| `count_word_in_book` | Count occurrences of a word or phrase in a book's full text |

#### Semantic Search (1 tool)

| Tool | Description |
|---|---|
| `count_related_words_in_book` | Find and count all words semantically related to a concept using embeddings |

#### Code Execution Analysis (2 tools)

| Tool | Description |
|---|---|
| `analyze_book_statistics` | Analyse arbitrary statistics or patterns in a book's text via Claude Code Execution |
| `generate_visualization` | Generate an interactive Plotly.js chart from book text analysis |

### Cursor Configuration (`.cursor/mcp.json`)

A `.cursor/mcp.json` file registers the server with Cursor under the name `book-app-tools`. The `BOOK_APP_USERNAME` environment variable identifies which user's favourites to operate on.

```json
{
  "mcpServers": {
    "book-app-tools": {
      "command": "node",
      "args": ["mcp-server.js"],
      "cwd": "/Users/navodanilakshi/Documents/book-app",
      "env": {
        "BOOK_APP_USERNAME": "<your-username>"
      }
    }
  }
}
```

## Files Changed

- **`mcp-server.js`** — new file; MCP server entry-point with all nine tool definitions
- **`.cursor/mcp.json`** — new file; Cursor MCP server registration config
- **`package.json`** — added `@modelcontextprotocol/sdk` and `zod` as runtime dependencies
