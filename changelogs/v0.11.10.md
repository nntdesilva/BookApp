# Version 0.11.10 - 2026-03-17

## Bug Fix: Strengthen Proactive Tool-Call Prevention Across All AI Tools

Extended the "never call tools proactively" restriction from word search (v0.11.8) to all AI tools. The chat assistant now answers general book questions (plot, themes, characters, author info) from its own knowledge and only invokes tools when the user explicitly requests an action. Added "ONLY call when the user explicitly asks... Never call proactively" to every tool description: favorites (add, remove, list, remove_all), word search, semantic search, book statistics, and visualization. Reorganized the meta-prompt structure (TAGGING, FAVORITES, OUTPUT FORMAT) for clarity.
