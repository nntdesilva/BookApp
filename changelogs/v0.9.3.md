# Version 0.9.3 - 2026-02-17

## New Feature: Clear Entire Favourites List via AI

Users can now clear their entire favourites list in one command by saying things like "remove all books from my favourites" or "clear my favourites list" in chat.

## What's New

### `remove_all_favorites` Tool

A new AI tool call that clears all books from the user's favourites list at once, rather than requiring individual removal by ISBN-13.

| Tool | Description |
|---|---|
| `remove_all_favorites` | Remove ALL books from the user's favourites list at once |

### How It Works

The AI recognises intent to clear the entire list and calls `remove_all_favorites` with no parameters. The `clearFavorites()` function in `favoriteService.js` sets the user's favourites array to empty and saves.

### Example

```
User: "Remove all books from my favourites"

AI Process:
1. Detects "clear all" intent
2. Calls remove_all_favorites()
3. favoriteService.clearFavorites(userId) empties the array
4. Returns: "Done — your favourites list has been cleared."
```
