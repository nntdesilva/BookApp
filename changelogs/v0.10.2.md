# Version 0.10.2 - 2026-02-26

## New Feature: Dark Mode

A persistent dark mode toggle has been added to all pages. Users can switch between light and dark themes at any time, and their preference is remembered across sessions.

## What's New

### Toggle Switch

A pill-shaped toggle switch is fixed in the top-right corner of every page (including login and signup). It displays a sun icon on the right and a moon icon on the left, with a sliding knob that moves to indicate the active mode.

### Persistent Preference

| User State | Storage |
|---|---|
| Logged in | `darkMode` field saved to MongoDB via `POST /api/dark-mode` |
| Guest | Preference stored in `localStorage` |

When a logged-in user returns, the page renders with their saved theme server-side (no flash of wrong theme). Guests retain their last choice via `localStorage` across visits.

### Theme Implementation

All colors in `public/css/style.css` were refactored from hardcoded values to CSS custom properties. A `[data-theme="dark"]` override block on the `<html>` element drives the full theme switch:

| Variable | Light | Dark |
|---|---|---|
| `--bg` | `#ffffff` | `#141414` |
| `--bg-card` | `#ffffff` | `#1c1c1c` |
| `--text-primary` | `#0a0a0a` | `#e0e0e0` |
| `--text-secondary` | `#6b6b6b` | `#8a8a8a` |
| `--border` | `#e0e0e0` | `#2a2a2a` |
| `--btn-bg` | `#0a0a0a` | `#e0e0e0` |
| `--btn-text` | `#ffffff` | `#141414` |

A small inline `<script>` in `<head>` applies the correct theme before the stylesheet renders, preventing any flash.

## Files Changed

- **`models/User.js`** — added `darkMode: Boolean` field (default `false`) to user schema
- **`routes/authRoutes.js`** — added `POST /api/dark-mode` endpoint to persist preference
- **`app.js`** — added global middleware to set `res.locals.currentUser = req.user` for all routes
- **`views/layouts/boilerplate.ejs`** — added toggle switch, flash-prevention script, and theme-aware `data-theme` attribute on `<html>`
- **`public/css/style.css`** — refactored all hardcoded colors to CSS custom properties; added `:root` light theme and `[data-theme="dark"]` dark theme overrides; added toggle switch styles
